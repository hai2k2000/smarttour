# Project Brief

SmartTour is a travel operations ERP for tour operators and travel agencies. The first implementation phase focuses on operational control rather than sales automation or AI.

## Primary Goal

Replace fragmented spreadsheets and chat-based coordination for tour operation teams with a centralized system for suppliers, tour programs, bookings, operation forms, costs, and supplier payments.

## MVP Workflow

```text
Create supplier
-> Create tour program
-> Create booking/departure
-> Create operation form
-> Assign services and suppliers
-> Track operation tasks
-> Record expected and actual costs
-> Create supplier payment requests
-> Review tour profit/loss
```
